## Project Overview
HomePro is a Django-based web application to manage service providers and service categories.

## Features
- User authentication (Login & Logout)
- Role-based access control
  - Admin: Full CRUD access
  - User: View only access
- CRUD operations using Django Admin
- Database integration using SQLite
- Clean and simple UI

## Roles
- Admin (is_staff): Can add, edit, delete service providers and categories
- Normal User: Can only view data

## Tech Stack
- Python
- Django
- SQLite
- HTML / CSS

## How to Run the Project
1. Activate virtual environment
2. Run migrations
3. Start server
